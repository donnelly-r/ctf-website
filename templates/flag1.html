Flag 1

Are you able to use the change directory linux command?

Instructions:

You are able to interact with the linux command line through the use of, you guessed it, commands. The first command you'll be introduced to is 'ls', which is shorthand for 'list directory contents'. This essentially outputs all the contents of any given folder on the machine.

Another such command is 'cd', shorthand for 'change directory'. In the context of linux, folders are referred to as directories, but essentially act as folders would on a Windows machine.

Your challenge for this particular flag is to use the 'cd' command to change directories to the 'Documents/flag1' directory. The format this command expects is as follows: 'cd <path>' where the path is the directory you wish to navigate to. From there, you must use the 'ls' command to list the contents of the flag1 directory. Doing so will retrieve the flag and you'll be able to move on to the next challenge!

If you get stuck or lost at any point, use the following command to reset back to the 'root' (top level) directory of the system:

cd ~

If you wish to 'go up' a directory level, use the command:

cd ..
