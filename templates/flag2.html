Flag 2

Can you use list a hidden file?

Instructions:

To begin this challenge please navigate to the correct directory by running 'cd ../flag2'

Now you must use an optional identifier known as a flag to modify the output of ls command to show hidden files. This is accessed by typing 'ls -[your_flag_here]' where the text '[your_flag_here]' should be replaced with a single character. Find the correct character to complete the challnege!

Help! I'm lost!

If you get stuck or lost at any point, use the following command to reset back to the 'root' (top level) directory of the system:

cd ~

Then run:

cd Documents/flag2

to get to the correct directory.

Context & Further Reading:

Here are some links to help you understand flags, as well as links to the optional flags that the ls command can use:

  • What are flags and how are they used?
  • All flags the ls command can use

Hints:

Ensure you are in the correct directory. Double check you are in the correct path by running 'pwd'. It should say '/home/user/Documents/flag2'.

Run the command 'ls -l'. See how this output differs from running just 'ls'. Try to change the 'l'' to another character to find the flag.

Run the command 'ls -a'. You should see the flag as a file beginning with the '.' character. This denotes a hidden file.

Check answer:

(the flag should be entered in the format 'flag{xxxx}' or 'xxxx' where x is a unique combination of characters)
